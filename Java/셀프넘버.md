# 💡 셀프넘버

## 💡 문제 설명

>양의 정수 `n`에 대해서 `d(n)`을 다음과 같은 **함수**라고 정의할 수 있다.
`d(n)`은 `n`을 입력 받아 `n`과 `n`의 각 자리수를 더하는 결과를 반환한다.<br>
>**예시)**  `d(75) = 75 + 7 + 5 = 87`
>이 때 75는 87의 **생성자**라고 하는데, **셀프 넘버**는 생성자가 없는 숫자를 의미한다. <br>
>100보다 작은 셀프 넘버는 총 13개가 있다.
>1, 3, 5, 7, 9, 20, 31, 42, 53, 64, 75, 86, 97
>정수를 입력 받아 셀프 넘버인지 확인하는 출력하는 프로그램을 작성하시오.

<br>

## 접근 방식

- 1부터 입력받은 양의 정수까지 생성자를 구한 다음 배열에 넣어 입력받은 수가 배열 안에 있으면 셀프넘버가 아니고, 없으면 셀프넘버를 출력하는 방식을 생각했다.
- 1부터 입력받은 수 까지 생성자를 구하기 위해서 for문을 사용했다.
- for문 안에서 각 자리수를 더하기 위해서 while문을 사용했다.
- 배열안에 값이 있는지 없는지 확인하기 위해 조건문을 사용해서 boolean 타입으로 확인했다.

<br>

## 💡 코드

~~~java
import java.util.Scanner;
    
    public class SelfNumber {
        public static void main(String[] args) {
            // 준비
            Scanner scanner = new Scanner(System.in);

            // 입력
            System.out.print("정수를 입력해주세요: ");
            int integerNumber = scanner.nextInt();

            int[] constructorArray = new int[integerNumber];

            int integerNumberSum = 0;

            // 처리
            for (int i = 0; i < constructorArray.length; i += 1) {
                integerNumberSum = 0;

                int plusNumber = i + 1;
                integerNumberSum += plusNumber;

                while (plusNumber > 0) {
                    integerNumberSum += plusNumber % 10;
                    plusNumber /= 10;
                }

                constructorArray[i] = integerNumberSum;
            }

            boolean isContainArray = false;

            for (int i = 0; i < constructorArray.length; i += 1) {
                if (constructorArray[i] == integerNumber) {
                    isContainArray = true;

                    break;
                }
            }

            // 출력
            if (!isContainArray) {
                System.out.println("셀프넘버입니다.");
            }

            if (isContainArray) {
                System.out.println("셀프넘버가 아닙니다.");
            }
        }
    }
~~~

<br>

## 💡 어려웠던 점

- 입력받은 수의 각 자리수를 더해서 생성자를 구한 다음 배열안에 넣는 과정과 입력받은 수가 배열에 있는지 없는지 확인하는 과정을 한번에 해결하기 위해 이중 for문을 이용하려 했던 점

- 각 자리수를 더한 후 자기 자신도 더해야 하는 코드를 생각하는데 어려움을 느낌

<br>

## 💡 배운점

- for문 안에서 조건문을 이용해 값을 비교하다 조건에 걸리면 break를 이용해 반복을 멈추게 하는 방법을 생각해냈다.
